알럿 버튼의 경우 텍스트 버튼(text, text + grey) 클래스 조합과 동일하나,
1. 텍스트 버튼의 사용이 알럿 팝업 외에 사용이 확인 되지 않아 차후 삭제될 수도 있으며,
2. 자주 사용하는 형태라 더 편한 형태로 예외 처리했습니다.
<div class="alert-button">
	<a href="#none" class="bt-base grey">알럿 팝업 버튼 1</a>
	<button class="bt-base">알럿 팝업 버튼 2</button>
</div>
